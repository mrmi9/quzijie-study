- [ ] 首页显示 500 题进度、今日答题、正确率、错题、收藏和五个模块卡片。

- [ ] Linux/操作系统、计网/STL 会先进入方向选择，两个方向进度独立。
- [ ] 首页普通未完成会话和 408 未完成试卷均能继续。

## 七学科普通练习

- [ ] 对 `cpp/linux/os/ds/network/stl/co` 分别创建章节与随机练习。
- [ ] 分别测试 5、10、20 题；章节不足所选题量时取全部且不重复。
- [ ] 单选和判断只能保留一个选择，多选重复点击可取消；未选择时确认禁用。
- [ ] 提交成功后选项锁定，正误、正确答案和解析正确显示。
- [ ] 最后一题完成后进入结果页，题数、正确数、错误数、正确率和章节表现一致。
- [ ] 代码片段在窄屏可横向滚动；若加入题图，能预览缩放且替代说明可见。
- [ ] 创建新普通练习后，旧会话提示 abandoned，首页只显示新会话。

## 网络、登录失效与幂等

- [ ] 普通题提交时模拟断网/超时/5xx，选择不丢失；恢复后重试只计一次。
- [ ] 答题、收藏和恢复时模拟 401，登录后返回原页且状态不丢失。
- [ ] 空题库、空错题、空收藏均显示明确空状态，不进入空白答题页。
- [ ] 将仓储切为 API 后所有页面仍走相同流程，页面源码中没有 `wx.request`。
- [ ] 真实微信模式连续登录两次得到同一业务用户；日志、客户端包和 Git 中没有 AppSecret、微信 code 或 Token。
- [ ] 刷新令牌成功轮换，旧刷新令牌再次使用返回 401，原业务请求只重放一次。

## 错题、掌握与收藏

- [ ] 错题重做答对后改为已掌握且历史保留；再次答错后恢复未掌握。
- [ ] 全局错题按七学科及掌握状态筛选准确。
- [ ] 答题反馈页收藏、模块收藏页取消和全局收藏数量同步。
- [ ] 全局错题/收藏必须选定学科后才能发起重练。

## 408 客观题模拟

- [ ] 时间归零自动交卷；断网时保留草稿并可重试；重复交卷不重复计分。
