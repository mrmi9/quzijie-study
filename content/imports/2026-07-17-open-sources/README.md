# 2026-07-17 开放资料原创题库批次

本批次计划新增 350 道题：C/C++、Linux、操作系统、数据结构、计算机网络、STL、计算机组成原理各 50 道。

## 内容边界

- 网络资料仅用于核验客观事实、标准版本和术语。
- 不抓取、不翻译、不改写任何现成题库，不复制规范、手册或教材的段落、示例、表格和插图。
- 所有中文题干、选项和解析均为本项目原创表达。
- ISO/IEC 草案、IETF RFC 等受版权保护的规范只记录链接与条款锚点；题目不构成规范文本副本。
- 首批新题不加入 408 模拟考试题池，避免未经体验版抽卷观察就改变正式考试难度分布。

## 复核门禁

每道题必须同时满足：

1. `factChecked`：事实与记录的来源锚点一致；
2. `answerChecked`：答案、选项和解析互相一致；
3. `copyrightChecked`：确认使用原创表述且未复制来源表达；
4. `duplicateChecked`：与现有 500 题及本批次其他题目无重复或高近似；
5. `reviewedBy`：复核者不能与 `authoredBy` 相同。

自动门禁命令：

```powershell
npm run validate:open-batch
```

## 发布物

- 七个学科 JSON：可审查的主数据；
- `sources.json`：来源、版本、许可与使用边界；
- `趣刷题喽-开放资料原创题库-2026-07-17.xlsx`：管理后台标准导入工作簿；
- `趣刷题喽-题库来源审查-2026-07-17.xlsx`：离线审查工作簿，不上传管理后台；
- `review-report.json`：数量、题型、难度、章节、来源、答案分布和重复度报告。

工作簿必须先进入管理后台暂存区，再由非提交人复核，最后由发布权限账号创建不可变发布批次。禁止直接写生产表。

管理后台导入文件严格保持“学科、章节、题目、选项、填空答案、媒体、说明”七张工作表。逐题来源锚点保存在独立审查工作簿和本目录 JSON 中；它们属于永久离线审计证据，不会被后台导入器静默当成题目字段。
